# Asteroids Game

## Project Documentation
The Asteroids game is a classic arcade game where the player controls a spaceship in space, avoiding and destroying asteroids. The goal is to survive as long as possible while accumulating points by shooting asteroids and enemies.

## Setup Instructions
1. **Clone the repository**:  
   ```bash
   git clone https://github.com/Indranil-Maity/asteroids.git
   cd asteroids
   ```  
2. **Install dependencies**:  
   Make sure to have Node.js and npm installed. Run:  
   ```bash
   npm install
   ```  
3. **Run the game**:  
   ```bash
   npm start
   ```  
   Open your browser and navigate to `http://localhost:3000` to play the game.

## Gameplay Controls
- **Arrow Up**: Move forward  
- **Arrow Down**: Move backward  
- **Arrow Left**: Rotate left  
- **Arrow Right**: Rotate right  
- **Spacebar**: Shoot

Enjoy the game!
